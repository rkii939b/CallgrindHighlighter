File Dialog

The File Dialog is used to open or save files in the Simple Text Viewer.

How to Use

To open a file using the file dialog:

  1. Click the "Open File" button in the main menu.
  2. Select the desired file from the dialog window that appears.
  3. Click "Open" to load the file into the viewer.

For more information on selecting files with wildcards, see Wildcard Matching.

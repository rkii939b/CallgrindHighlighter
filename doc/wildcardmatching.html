Wildcard Matching

This section explains how to use wildcard patterns to filter files in the Simple Text Viewer.

Supported Wildcards

  • *: Matches any sequence of characters (e.g., *.txt will match all text files).
  • ?: Matches any single character (e.g., file?.txt will match file1.txt, file2.txt, etc.).

Example

To find all files starting with "doc" and ending with ".txt", use the pattern doc*.txt.

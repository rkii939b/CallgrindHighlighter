Find File

This section explains how the "Find File" functionality works in Simple Text Viewer.

Usage

To find a specific file in the viewer, follow these steps:

  • Click the "Find File" button in the main toolbar.
  • Enter the search term in the input field and click "Search."
  • The viewer will highlight matching files in the list.

Advanced Features

The "Find File" dialog supports advanced filtering options using wildcards. Refer to Wildcard Matching for more details.
